# RAG-Based_Sales_Data_Analysis

Retrieval-Augmented Generation (RAG) system to analyze a real-world sales dataset. Using a vector database to store and retrieve relevant data, and integrate a LLM to answer analytical questions about sales trends, patterns, and insights.

## Dataset

Used dataset is [Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final/data). Dataset contains Information related to Sales, Profits and other interesting facts of a Superstore giant.
